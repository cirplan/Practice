### 231. 2的幂

> 题目

给定一个整数，编写一个函数来判断它是否是 2 的幂次方。

示例 1:
```
输入: 1
输出: true
解释: 20 = 1
```

示例 2:
```
输入: 16
输出: true
解释: 24 = 16
```

示例 3:
```
输入: 218
输出: false
```

> 思路

2的幂次方一定 > 0 即 n > 0，n & (n - 1) === 0，2的幂次方用二进制表示，只有一个1，后面0。2的幂次方-1用二进制，原来1变0，后面1

> 代码

```js
/**
 * @param {number} n
 * @return {boolean}
 */
var isPowerOfTwo = function(n) {
    return n > 0 && (n & (n - 1)) === 0
};
```

时间复杂度 : O(1)。

空间复杂度 : O(1)。
